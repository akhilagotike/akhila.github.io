Lights Out (3×3)

Addendum

Process & Creative Choices: The game was implemented with modular and responsive design. I added a difficulty selector that changes the number of random toggles applied to the board.

Challenges: The biggest challenge was generating a solvable board that still varied each time. I solved this by simulating real game moves on a clean board instead of toggling tiles randomly.

Insights: I gained a deeper understanding of DOM manipulation, CSS grid layout, and modular event-based game logic.
