Lights Out

Addendum

This game was built using JavaScript and CSS grid. Depending on the level selected, the grid size changes (3x3, 5x5, 7x7). The starting configuration is created by making random valid toggles on an all-lit board, ensuring it's always solvable.

The biggest challenge was making the game flexible to work for different grid sizes and random setups. I used a loop to simulate player moves during board generation to overcome this.

I learned how to use dynamic layout rendering and improved my skills in DOM scripting and event-driven logic through this project.
