Treasure Hunt

Addendum

This project is an interactive "Choose Your Own Adventure" story created using HTML, CSS, and JavaScript. The story follows Gautham Kumar Varma on a treasure hunt, where the player makes choices that change the outcome.

The JavaScript dynamically updates the story text, images, and choices. A restart button is included on every page so players can easily start over at any time.

Challenges Faced:

Restart Function: Made sure the restart button appears on all pages.

Styling Issues: Adjusted CSS for better readability and user experience.

This project helped improve my JavaScript and CSS skills while making an engaging interactive story.
